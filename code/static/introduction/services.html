Services

Help The Global Anti-Epidemic

Our work is mainly to collect data through web crawlers, generate epidemic dynamic map or other data visualization products, and provide some basic popular science information about new coronavirus for the public

Base map making

using the Gaud API to obtain the boundary data of administrative divisions then draw the map by the use of some extention bags, echarts-china-provinces-pypkg ,etc


Web Crawler

Using the extension package jsoup in java or other methods to visit the API of publishing epidemic data in news websites, such as Tencent news api, then we can get the JSON file

Data integration

Using JSON fragment converter to analyze the historical data of epidemic situation, then manually revise it according to your map scope, get a set of data tables


Data visualization

First make the data corresponding work. Different data ranges are marked with different colors to ensure visual effect. Then use pyecharts library to form the map you need

popularization of science

present the basic information and latest scientific developments on covid-19. We will ensure the authenticity of all the information
